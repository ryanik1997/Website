───────────────────────────────────────────────────────────────
1. CHUẨN BỊ TRƯỚC KHI CHAT
───────────────────────────────────────────────────────────────

Bạn cần có:
  • PDF đề Listening (hoặc ảnh/screenshot từng Section)
  • Answer Key (đáp án 1–40)
  • File audio ~30 phút (đổi tên listening.mp3 khi đóng ZIP)
  • (Khuyến nghị) Ảnh UI tham chiếu trong Giaodien/ — để ChatGPT khớp layout
  • Choose TWO: Giaodien/two-choice.jpg — Answer Key A/E + UI 2 ô (Part 1–3)

Tải mẫu JSON trong app:
  Luyện thi → IELTS → Import thủ công Listening → nút tải theo Part/dạng
  Hoặc lấy sẵn: Tainguyen/templates/ielts-listening-*.json

2. CHỌN CÁCH LÀM — 2 WORKFLOW
───────────────────────────────────────────────────────────────

┌─────────────────────────────────────────────────────────────┐
│ A. LÀM TỪNG PART (khuyến nghị — đề trộn nhiều dạng)        │
│    4 lần chat → ghép 4 parts[] → 1 exam.json                │
├─────────────────────────────────────────────────────────────┤

└─────────────────────────────────────────────────────────────┘

**Workflow A — từng Part (đề trộn a6 + c3 + d2…):**

  Bước 1  Nhìn PDF Part 1 → chọn dạng (a2/a3/a4/a5) → tải JSON mẫu
  Bước 2  ChatGPT: đính kèm Prompt-Part1.txt + PDF P1 + Key 1–10 + mẫu JSON
  Bước 3  Lưu kết quả → part1.json (object parts[0])
  Bước 4  Lặp Part 2 (a6–a14), Part 3 (c1–c7), Part 4 (d1–d3)
  Bước 5  Ghép 4 part → exam.json (mục 5 bên dưới)
  Bước 6  ZIP: exam.json + listening.mp3 (+ map.jpg / diagram.jpg nếu có)
  Bước 7  Import trong app → Preview → Lưu & làm bài

**Workflow B — cả đề một lần:**

  Đính kèm: Prompt-IELTS-Listening-Cam9-Cam20.txt + PDF + Key + ielts-listening-full.json
  Yêu cầu ChatGPT xuất exam.json đủ 4 parts, 40 câu.
  Phù hợp Cam9/Cam20 Test 1 hoặc đề có cấu trúc tương tự builtin.

───────────────────────────────────────────────────────────────
3b. CHOOSE TWO — "Which TWO …?" (Part 1–3, không có Part 4)
───────────────────────────────────────────────────────────────

Tham chiếu: Giaodien/two-choice.jpg (Cam20 Part 2, câu 17–20)

Trên Answer Key:
  17  A/E    → câu 17 đúng nếu chọn A hoặc E
  18  A/E    → câu 18 đúng nếu chọn A hoặc E
  19  C/E    → nhóm khác: C hoặc E
  20  C/E

Trên UI app:
  • Danh sách 5 lựa chọn A–E (có ô checkbox trang trí)
  • 2 hàng đánh số 17 và 18 — mỗi hàng pill A B C D E để chọn

Khi viết exam.json cho ChatGPT:
  • Tách 2 câu matching liên tiếp (không gộp 1 câu)
  • Cùng prompt: "Which TWO things does Heather explain about kilns?"
  • Cả 2 câu: "answer": "A/E"
  • options nhãn đầy đủ — xem p2-a10-template.json

ChatGPT đọc Answer Key "17 A/E" → ghi answer "A/E" cho câu 17 VÀ câu 18.

─── PROMPT CHUNG (đầu mỗi lần chat Part) ───

```
Bạn là trợ lý tạo exam.json IELTS Listening cho web app "Ryan English".

Đọc KỸ file đính kèm: HDSD/Prompt-IELTS-Listening-Part[N].txt
Copy CẤU TRÚC từ file JSON mẫu đính kèm — chỉ đổi nội dung và đáp án.

## Tôi đính kèm
- Prompt Part [N] (quy tắc JSON)
- PDF / ảnh Section [N]
- Answer Key câu [range]
- File mẫu JSON: [tên file]
- (Nếu có) ảnh UI Giaodien/…
- (Nếu có) map.jpg / diagram.jpg

## Đề Part [N] này gồm (theo thứ tự câu trên PDF):
[Ghi từng đoạn — ví dụ Part 2:]
- Questions 11–16: notes → a7
- Questions 17–18: MC → a7
- Questions 19–20: Choose TWO → a7

## Yêu cầu output
1. Xuất 1 object JSON part (partNumber: [N], đủ 10 câu).
2. questions[] ĐÚNG THỨ TỰ trên đề.
3. gap-fill: mỗi gap có trong notePassage / noteTable / notePassageSections.
4. Choose TWO: 2 câu matching, options A–E NHÃN ĐẦY ĐỦ, answer "X/Y".
5. Section header (sectionRange, sectionInstruction, sectionTitle) trên câu ĐẦU mỗi nhóm.
6. wordLimit khớp instruction đề; explanation tiếng Việt ngắn.
7. audioFile: "listening.mp3" trên Part.
8. Chỉ JSON thuần — không markdown, không giải thích thêm.

Bắt đầu.
```

─── Gợi ý điền nhanh theo Part ───

Part 1 — range 1–10 · prompt Part1.txt · mẫu p1-form / p1-table / p1-mixed-a4 / a5
Part 2 — range 11–20 · prompt Part2.txt · mẫu p2-a6…a14 · thêm map/diagram nếu a6/a9/a12/a14
Part 3 — range 21–30 · prompt Part3.txt · mẫu p3-c1…c7 · notePassageSections nếu c1/c2
Part 4 — range 31–40 · prompt Part4.txt · mẫu p4-d1/d2/d3 · notePassage đủ static+section+gap

───────────────────────────────────────────────────────────────
5. GHÉP 4 PART → exam.json
───────────────────────────────────────────────────────────────

Sau 4 lần chat, dán prompt ghép:

```
Ghép 4 object part JSON đính kèm thành 1 file exam.json import IELTS Listening.

Cấu trúc:
{
  "version": 1,
  "title": "IELTS Listening — Cambridge [N] Test [M]",
  "durationMinutes": 30,
  "bandHint": "IELTS · Cambridge [N] · 4 parts · 40 câu",
  "examType": "ielts",
  "examMode": "practice",
  "parts": [ part1, part2, part3, part4 ]
}

Yêu cầu:
- parts[0].partNumber = 1 … parts[3].partNumber = 4
- Tổng 40 câu, số câu 1→40 liên tục, không trùng
- Giữ nguyên nội dung từng part (không tự sửa đáp án)
- Mỗi part: audioFile "listening.mp3"
- Chỉ xuất JSON thuần
```

Hoặc tự ghép tay: mở 4 file → copy từng object vào mảng `parts[]`.
