### Bắt buộc: `notePassage` cho Part gap-fill (form / notes)

IELTS form/lecture notes có **nhiều dòng chữ tĩnh** xen giữa các ô trống. Phải khai báo **đầy đủ thứ tự** trong `notePassage` — app render y hệt đề gốc.

```json
"notePassage": [
  { "type": "static", "text": "Example: Work at: a restaurant" },
  { "type": "gap", "number": 1 },
  { "type": "static", "text": "Number of hours per week: 12 hours" },
  { "type": "static", "text": "Would need work permit" },
  { "type": "gap", "number": 2 },
  { "type": "section", "text": "Extra benefits" },
  { "type": "static", "text": "a free dinner" },
  { "type": "gap", "number": 5 }
]
```

| Block | Ý nghĩa |
|-------|---------|
| `static` | Dòng chữ không có ô trống |
| `section` | Tiêu đề mục (Personal information, Parasites…) |
| `example` | Dòng Example (in nghiêng — dùng với `notePassageLayout: "form"`) |
| `gap` | Ô trống — `number` trùng `questions[].number` |

Mỗi câu `gap-fill` trong `questions` **phải xuất hiện đúng 1 lần** trong `notePassage` hoặc `noteTable`. Preview import sẽ cảnh báo nếu thiếu.

### `noteTable` cho Part 1 bảng (table-completion)

```json
"notePassageLayout": "table",
"noteTable": {
  "headers": ["Name of restaurants", "Location", "Reason for recommendation", "Other comments"],
  "rows": [
    {
      "cells": [
        [{ "type": "static", "text": "The Junction" }],
        [{ "type": "static", "text": "Greyston Street, near the station" }],
        [
          { "type": "static", "text": "Good for people who are especially keen on " },
          { "type": "gap", "number": 1 }
        ],
        [
          { "type": "static", "text": "Quite expensive" },
          { "type": "break" },
          { "type": "static", "text": "The " },
          { "type": "gap", "number": 2 },
          { "type": "static", "text": " is a good place for a drink" }
        ]
      ]
    }
  ]
}
```

Cell blocks: `static` | `gap` | `break` (xuống dòng trong cùng ô).

### `noteTables[]` — Part 1 nhiều bảng + MC ở giữa (a4)

Khi Part 1 có **hai bảng** xen Choose TWO, dùng mảng `noteTables` thay vì một `noteTable`:

```json
"notePassageLayout": "table",
"noteTables": [
  {
    "gapNumbers": [1, 2, 3, 4],
    "instruction": "Questions 1–4\nComplete the table below…",
    "headers": ["Name of centre", "Doctor's name", "Advantage"],
    "rows": [ /* … */ ]
  },
  {
    "gapNumbers": [7, 8, 9, 10],
    "title": "Talks for patients at Shore Lane Health Centre",
    "instruction": "Questions 7–10\nComplete the table below…",
    "headers": ["Subject of talk", "Date/time", "Location", "Notes"],
    "rows": [ /* … */ ]
  }
],
"questions": [
  /* gap-fill 1–4 */,
  /* matching 5–6 Choose TWO, answer "A/E" */,
  /* gap-fill 7–10 */
]
```

**Thứ tự `questions` bắt buộc:** gap-fill liên tiếp → MC/matching → gap-fill (app tách segment tự động).

### Part 1 MC + gap-fill (a5)

```json
"questions": [
  { "number": 1, "type": "multiple-choice", "prompt": "…", "options": […], "answer": "B" },
  /* … MC 2–6 … */
  { "number": 7, "type": "gap-fill", "prompt": "…", "answer": "…", "wordLimit": 1 }
],
"notePassage": [
  { "type": "section", "text": "Paxton Nature Reserve" },
  { "type": "static", "text": "• Paxton is a good place for seeing rare" },
```

`notePassage` chỉ cần khối cho câu gap-fill (7–10); MC 1–6 không cần notePassage.

## Choose TWO — "Which TWO …?" (Part 1–3, không Part 4)

**Tham chiếu UI + Answer Key:** `Giaodien/two-choice.jpg` (Cam20 P2 câu 17–20)

Answer Key ghi `17 A/E`, `18 A/E` nghĩa là:
- Hai câu 17 và 18 thuộc **một nhóm** Choose TWO (cùng câu hỏi trên đề)
- Mỗi ô đều đúng nếu học sinh chọn **A hoặc E** (thứ tự không bắt buộc: 17=A+18=E hoặc 17=E+18=A)
- Nhóm tiếp theo có thể khác: `19 C/E`, `20 C/E`

Quy tắc JSON:
- Tách **2 câu** `type: "matching"` liên tiếp (vd. 17 + 18), **cùng prompt**
- `options`: 5 mục A–E, **nhãn đầy đủ** — vd. `{ "id": "A", "label": "What their function is." }`
  Không chỉ `"label": "A"` (UI hiện "A A")
- `answer`: `"A/E"` trên **cả 2 câu** — app chấp nhận A hoặc E cho từng ô
- Câu đầu nhóm: `sectionRange` + `sectionInstruction`
- Dùng helper `choose_two()` trong `scripts/build-ielts-listening-tests.py`

Xuất hiện: Part 1 a4 (5–6) · Part 2 a7/a10/a13 · Part 3 c3/c5
