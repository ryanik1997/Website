## Schema Part 4

```json
{
  "partNumber": 4,
  "rangeLabel": "Questions 31–40",
  "instruction": "Complete the notes below. Write ONE WORD ONLY for each answer.",
  "passageTitle": "Reclaiming urban rivers",
  "audioFile": "listening.mp3",
  "notePassage": [
    { "type": "section", "text": "Historical background" },
    { "type": "static", "text": "Nearly all major cities were built on a river." },
    { "type": "gap", "number": 31 },
    { "type": "gap", "number": 32 }
  ],
  "questions": [
    {
      "number": 31,
      "type": "gap-fill",
      "prompt": "Pollution source:",
      "gapLead": "pollution from",
      "gapTrail": "on the river bank",
      "wordLimit": 1,
      "answer": "factories",
      "explanation": "Nhà máy."
    }
  ]
}
```

## Thứ tự `questions[]`

Part 4 chỉ có **10 câu gap-fill liên tiếp** (31→40). Không cần sectionRange trên từng câu
(trừ khi đề PDF có nhiều nhóm instruction — hiếm ở P4).

## notePassage blocks

| type    | Ý nghĩa                                      |
|---------|----------------------------------------------|
| static  | Dòng chữ cố định (mô tả bài giảng)           |
| section | Tiêu đề mục (Parasites, Historical background) |
| gap     | Ô trống — number = questions[].number        |

**Bắt buộc:** Khai báo **đầy đủ thứ tự** static + section + gap như trên đề PDF.
App render y hệt đề gốc — thiếu dòng static = layout sai.

## gapLead / gapTrail

Trên **mỗi câu** `gap-fill` trong `questions[]`:

- `gapLead` — chữ ngay **trước** ô trống (có thể gồm bullet `•`)
- `gapTrail` — chữ ngay **sau** ô trống

Dùng khi chữ trước/sau gap nằm **cùng dòng** với ô trống (inline), không phải block `static` riêng.

**Ví dụ d1 (bullet sub-item):**
```json
{ "number": 36, "gapLead": "• were all", "answer": "healthy" }
{ "number": 37, "gapLead": "• were not in a", "answer": "group" }
```
→ `notePassage` có 2 gap liên tiếp (36, 37) sau dòng static "The Bahamas…".

**Ví dụ d2 (câu tách 39–40):**
```json
{ "number": 39, "gapLead": "goods could be transported by large freight barges and electric", "answer": "bikes" }
{ "number": 40, "gapLead": ", or in the future, by", "answer": "drone" }
```
→ Hai gap liên tiếp; gapLead câu 40 bắt đầu bằng dấu phẩy nối từ câu trước.

## wordLimit

| Instruction trên đề              | wordLimit |
|----------------------------------|-----------|
| ONE WORD ONLY                    | 1         |
| NO MORE THAN TWO WORDS           | 2         |
| NO MORE THAN THREE WORDS AND/OR A NUMBER | 3 |

`instruction` ở cấp Part phải khớp đề PDF.

## Không dùng

- `notePassageSections[]` — chỉ Part 3 (nhiều khối note tách instruction)
- `noteTable` / `noteTables[]` — Part 4 lecture notes không phải bảng
- `notePassageLayout: "form"` — Part 4 dùng list (bỏ qua field hoặc không khai báo)
