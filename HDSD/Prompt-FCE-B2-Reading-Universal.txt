## ⚠ SCHEMA BẮT BUỘC — app từ chối nếu sai

KHÔNG dùng: part, title (trong part), questions (ngoài questionGroups), id, question, options {label,text}.

Mỗi part PHẢI có:
- partNumber, rangeLabel, passageTitle
- passage[] — mảng object (KHÔNG string)
- questionGroups[] — MẢNG; mọi câu hỏi nằm TRONG questionGroups[].questions

Mỗi câu PHẢI có:
- number (KHÔNG id), type, prompt (KHÔNG question)

## ⚠ JSON HỢP LỆ — BẮT BUỘC (app từ chối nếu JSON.parse fail)

`exam.json` phải là **JSON chuẩn RFC 8259** — app gọi `JSON.parse()` ngay khi import.
**Một lỗi cú pháp → toàn bộ đề không import được** (không có cảnh báo từng câu).

### 1. Dấu ngoặc kép `"` trong chuỗi — LỖI PHỔ BIẾN NHẤT

Mọi giá trị text (`prompt`, `label`, `text`, `explanation`…) là chuỗi JSON.
**Bên trong chuỗi KHÔNG được có dấu `"` ASCII trần** — phải escape `\"` hoặc **bỏ hẳn / thay bằng từ khác**.

| SAI — JSON hỏng | ĐÚNG |
|-----------------|------|
| `"explanation": "... cụm "rather than" ..."` | `"explanation": "... cụm rather than ..."` |
| `"answer": "b (rather)"` | `"answer": "b"` |
| `"features": [{ "name": "A. Through endless tries…" }]` | `"name": "Through endless tries…"` |
| Đoạn có lời thoại `"Look, if you can…"` trong passage | Escape: `\"Look, if you can…\"` hoặc dùng nháy đơn trong PDF |

**Quy tắc explanation (tiếng Việt):** KHÔNG bọc từ EN bằng `"..."` — ghi thẳng hoặc dùng nháy đơn `'rather than'`.

### 2. Trường `answer` — chỉ giá trị chấm được, KHÔNG ghi chú

| Loại câu | `answer` ĐÚNG | SAI |
|----------|---------------|-----|
| multiple-choice (Part 1, 5) | `"a"` / `"b"` / `"c"` / `"d"` — chữ thường, 1 ký tự | `"B"`, `"c (best)"` |
| matching-features (Part 6) | `"a"` … `"g"` — chữ thường, 1 ký tự | `"G - sentence text"` |
| matching-features (Part 7) | `"a"` … `"d"` — chữ thường, 1 ký tự | `"Paragraph A"`, `"Section A"` |
| gap-fill Part 2 (open cloze) | **một từ**: `"where"`, `"which"` | `"which (that)"` |
| gap-fill Part 3 (word formation) | **một từ** biến đổi: `"producer"` | `"PRODUCT → producer"` |
| gap-fill Part 4 (key word) | **cụm 2–5 từ** thiếu: `"a good idea to go"` | ghi KEY word vào answer |
| đáp án kép (key có 2+ cách) | `"a good idea to go|a good idea of going"` hoặc `từ1/từ2` | `"idea to go (or of going)"` |
| writing-task Part 8–9 | `""` (chuỗi rỗng) | `"N/A"`, bỏ trường answer |

Giải thích / biến thể → `explanation` hoặc `/` `|` trong `answer` — **không** ghi chú ngoặc trong answer.

### 3. Các lỗi JSON khác — KHÔNG được phép

- **Hai JSON dính liền** — `}{` giữa 2 object (append test cũ)
- Trailing comma, comment `//`, smart quotes `"` `"`
- Single quotes bọc key/value: `'answer': 'a'`

### 4. Tên file ảnh — so sánh KET / PET / FCE

| Level | Part Writing | Quy ước ảnh |
|-------|--------------|-------------|
| KET A2 | P6 email | 1 ảnh `part6-page.jpg` |
| KET A2 | P7 story | **3 ảnh** `part7-p1.jpg` … `part7-p3.jpg` |
| PET B1 | P7 message | 1 ảnh `part7-page.jpg` |
| PET B1 | P8 story | **1 ảnh** `part8-page.jpg` (3 khung trong 1 file) |
| **FCE B2** | **P8 essay** | **1 ảnh** `part8-page.jpg` |
| **FCE B2** | **P9 story** | **1 ảnh** `part9-page.jpg` (3 khung trong 1 file) — **KHÔNG** `part9-p1…p3` |

### 5. Tự kiểm tra TRƯỚC KHI xuất ZIP (bắt buộc)

1. **JSON.parse** toàn bộ `exam.json` — nếu fail thì sửa, không giao file
2. File chỉ có **1** object JSON (không có `}{` thừa)
3. Đếm: **9 parts · 54 mục** (Q1–52 Reading + Q53–54 Writing) — hoặc 7 part + ảnh Writing
4. Part 1: 8 câu MC × 4 options — KHÔNG gap-fill
5. Part 6: đủ 7 câu A–G trong passage + features; 6 gap 37–42
6. Part 7: options label **Paragraph A** … **Paragraph D**; 10 câu 43–52
7. Part 8–9: type `writing-task`, `minWords: 140`, `answer: ""`
8. Part 8–9 (ảnh): passage[] có **đúng 1** block `imageFile`
9. Parts 1–7: `passage[].text` có nội dung (Highlight)
10. `imageFile` trong JSON = tên file thật trong ZIP
11. `durationMinutes`: **80** (không 75)

## Part 1 (Q1–8) — Multiple-choice cloze

App: gap `(1)` … `(8)` **trong passage** — bấm gap → chọn 1 trong 4 từ A/B/C/D.

```json
{
  "partNumber": 1,
  "rangeLabel": "Questions 1–8",
  "passageTitle": "What is genealogy?",
  "passage": [
    { "text": "What is genealogy?" },
    { "text": "Genealogy is a (0) …..... of history. It concerns family history, (1) ........ than the national or world history studied at school. It doesn't merely involve drawing a family tree, however – tracing your family history can also (2) ........ in learning about your roots and your identity." }
  ],
  "questionGroups": [{
    "range": "Questions 1–8",
    "instruction": "For each question, choose the correct answer.",
    "type": "multiple-choice",
    "questions": [
      {
        "number": 1,
        "type": "multiple-choice",
        "prompt": "Gap (1)",
        "options": [
          { "id": "A", "label": "instead" },
          { "id": "B", "label": "rather" },
          { "id": "C", "label": "except" },
          { "id": "D", "label": "sooner" }
        ],
        "answer": "b",
        "explanation": "rather than — hơn là."
      }
    ]
  }]
}
```

- Giữ `(1) ........` … `(8) ........` (**8 chấm**) trong passage
- Example `(0)` có đáp án trong passage — **KHÔNG** tạo câu hỏi số 0
- `options[].label` = **từ tiếng Anh** từ đề (không phải Option A)
- **KHÔNG** dùng `gap-fill` — Part 1 luôn `multiple-choice` + 4 options

PDF có dòng **Word stems** + passage có `(17) ........`.
App: split pane — passage trái + **Keyword List** phải (stem từ `prompt`).

```json
{
  "partNumber": 3,
  "rangeLabel": "Questions 17–24",
  "passageTitle": "[tiêu đề bài]",

    { "text": "Word stems: COMMON · PRODUCT · ILL · EFFECT · SCIENCE · ADD · PRESS · DISADVANTAGE · SPICE" },
    { "text": "[đoạn có (17) ........ … (24) ........]" }
  ],
  "questionGroups": [{
    "range": "Questions 17–24",
    "instruction": "Write ONE word for each gap.",
    "type": "gap-fill",
    "questions": [
      {
        "number": 17,
        "type": "gap-fill",
        "prompt": "Gap (17) — PRODUCT",
        "options": [],
        "answer": "producer",
```

## Part 4 (Q25–30) — Key word transformation

Mỗi câu: câu gốc + **KEY WORD** + câu đệm có gap `…`.
App: danh sách câu — câu nguồn, KEY in đậm, câu đích có ô gap.

```json
{
  "partNumber": 4,
  "rangeLabel": "Questions 25–30",
  "passageTitle": "Part 4 — Key word transformation",
  "passage": [
    { "text": "Complete the second sentence using the word given. Use between two and five words, including the word given." }
  ],
  "questionGroups": [{
    "range": "Questions 25–30",
    "instruction": "Write only the missing words. Use between two and five words, including the word given.",
    "type": "gap-fill",
    "questions": [
      {
        "number": 25,
        "type": "gap-fill",
        "prompt": "Joan was in favour of visiting the museum. IDEA → Joan thought it would be … to the museum.",
        "options": [],
        "answer": "a good idea to go",
        "explanation": "a good idea to go"
      },
      {
        "number": 28,
        "type": "gap-fill",
        "prompt": "I knocked for ages at Ruth's door but I got no reply. LONG → I … knocking at Ruth's door but I got no reply.",
        "options": [],
        "answer": "spent a long time",
        "explanation": "spent/took/was a long time — answer key chấp nhận nhiều cách."
      }
    ]
  }]
}
```

- `prompt` dùng ` → ` hoặc `->` — app tách câu 1 / KEY / câu 2
- `answer` = **cụm từ thí sinh điền** (phần gap)
- Nhiều đáp án key → `"spent a long time|took a long time|was a long time"`

- Intro (tiêu đề, byline) = block `text` **không** label
- 10 câu 43–52; một paragraph có thể trả lời **nhiều câu**
