## Thứ tự `questions[]` (app tách segment)

```
gap-fill → multiple-choice → matching (letter) → choose-two → diagram → map
```

**Ví dụ a7:** gap 11–16 → MC 17–18 → Choose TWO 19–20  
**Ví dụ a8:** MC 11–13 → matching 14–18 → gap (bảng) 19–20  
**Ví dụ a10:** MC 11–16 → Choose TWO 17–18 → Choose TWO 19–20

## Choose TWO — "Which TWO …?" (a7, a10, a13)

**Tham chiếu:** `Giaodien/two-choice.jpg` (Cam20 P2 câu 17–20) · mẫu `p2-a10-template.json`

Trên Answer Key (Cam20 Test 1 P2):
| Câu | Answer Key | exam.json (cả 2 câu nhóm) |
|-----|------------|---------------------------|
| 17–18 | 17 A/E, 18 A/E | `"answer": "A/E"` |
| 19–20 | 19 C/E, 20 C/E | `"answer": "C/E"` |

Cam9 Test 1 P2 (a7): câu 19–20 → `"answer": "A/E"`.

**UI app:** danh sách 5 options A–E + 2 ô (vd. 17 và 18), mỗi ô chọn 1 chữ cái.
Mỗi ô đúng nếu chọn **một trong hai** chữ cái đúng (A hoặc E). Thứ tự giữa 2 ô không bắt buộc.

**KHÔNG** nhầm với letter matching (a8, a11) — Choose TWO có prompt dài "Which TWO…",
5 options nhãn đầy đủ; letter matching có prompt ngắn (tên người) và đáp án 1 chữ cái.

```json
{
  "number": 17,
  "type": "matching",
  "prompt": "Which TWO things does Heather explain about kilns?",
  "options": [
    { "id": "A", "label": "What their function is." },
    { "id": "B", "label": "When they were invented." },
    { "id": "C", "label": "Ways of keeping them safe." },
    { "id": "D", "label": "Where to put one in your home." },
    { "id": "E", "label": "What some people use instead of one." }
  ],
  "answer": "A/E",
  "explanation": "Chức năng lò (A) và thay thế (E).",
  "sectionRange": "Questions 17 – 18",
  "sectionInstruction": "Choose TWO letters, A–E."
},
{
  "number": 18,
  "type": "matching",
  "prompt": "Which TWO things does Heather explain about kilns?",
  "options": [ /* GIỐNG HỆT câu 17 */ ],
  "answer": "A/E",
  "explanation": "Cặp đáp án 17–18: A và E."
}
```

Quy tắc:
- **2 câu** `matching` **liên tiếp**, **cùng prompt**, **cùng options**
- `answer` **giống nhau** trên cả 2 câu — đọc từ Answer Key (vd. 17 A/E → `"A/E"`)
- `options`: nhãn đầy đủ — KHÔNG `"label": "A"`
- `sectionRange` + `sectionInstruction` trên câu **đầu** nhóm
- a10 có **2 nhóm** Choose TWO liên tiếp (17–18 rồi 19–20) — mỗi nhóm 2 câu, answer riêng

## Letter matching (a8, a11, a13, a9 phần 14–18) — KHÁC Choose TWO

```json
{
  "number": 14,
  "type": "matching",
  "prompt": "Simon",

    { "id": "A", "label": "advertising" },
    { "id": "B", "label": "animal care" }
  ],
  "answer": "C",
  "sectionRange": "Questions 14 – 18",
  "sectionInstruction": "What is currently the main area of work… Choose FIVE answers A–H.",
  "sectionTitle": "AREA OF WORK"
}
```

- Nhiều câu matching, **options giống nhau** mọi câu
- `prompt` = tên người / item ngắn

## Map labeling (a6, a12, a14)

- Part: `"imageFile": "map.jpg"`
- Câu đầu nhóm map: `"mapLabel": true`
- `options`: A–I, label có thể chỉ chữ cái
- `prompt`: tên địa điểm (bird hide, bridge foundations…)

## Diagram labeling (a9)

- Part: `"imageFile": "diagram.jpg"`
- Câu 11–13: `"diagramLabel": true`
- `options`: A–E với nhãn đầy đủ (electricity indicator…)
