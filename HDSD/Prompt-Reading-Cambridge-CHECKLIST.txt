## 1) JSON hợp lệ

- [ ] `JSON.parse` PASS — 1 object
- [ ] `version: 1`
- [ ] `examTrack`: `"cambridge"`
- [ ] `cambridgeLevel`: `"a2"` | `"b1"` | `"b2"` | `"c1"` | `"c2"`
- [ ] `title` có **Book N** + **Test N**  
  VD: `"KET A2 Reading — Book 1 — Test 2"`
- [ ] `durationMinutes` đúng (KET RW 60, PET RW 75, FCE/CAE/CPE theo prompt)
- [ ] Schema **Reading**: `parts[].questionGroups[].questions[]`  
  (không dùng Listening `questions[]` thẳng trong part)
- [ ] Không `"` ASCII trần trong explanation

## 4) questionGroups & type

- [ ] Mỗi group: `range`, `instruction`, `type`, `questions[]`
- [ ] Type hợp lệ: tfng | ynng | multiple-choice | matching-* | gap-fill | summary-completion | sentence-completion | writing-task
- [ ] MC: options `id` A/B/C (hoặc a/b/c) + `answer` một ký tự
- [ ] matching-headings: `headings[]` khớp `answer`
- [ ] writing-task: `minWords`, `answer` có thể `""`

## 5) answer

- [ ] Đối chiếu **từng câu Reading** với answer-key
- [ ] MC: `"a"` / `"b"` / `"c"` — không `"B (best)"`
- [ ] TFNG / YNNG: đúng enum app (`true`/`false`/`not-given` hoặc YES/NO/NOT GIVEN theo prompt level)
- [ ] gap-fill: một từ; kép: `a/one`
- [ ] Writing: không bịa key — để trống + chấm AI sau
- [ ] ZIP có `answer-key.txt` / `.pdf` → app chỉ **bổ sung** câu JSON đang trống
