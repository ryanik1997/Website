## Part 2 (Q6–10) — Matching markets A–H ⚠ QUAN TRỌNG

App hiển thị:
- **Cột trái:** intro + danh sách A–H (tên market + mô tả đầy đủ)
- **Cột phải:** 5 câu mô tả người + pills chọn A–H (không lặp danh sách)

**ĐÚNG — passage (đủ 8 market, label = chữ cái):**
```json
"passage": [
  { "text": "The people below all want to visit a city market. Decide which market would be the most suitable for the people below." },
  { "label": "A", "text": "Beckfield Market — This market's world-famous for second-hand camera equipment…" },
  { "label": "B", "text": "Rosewell Hill — Our market's in an amazing building…" },
  { "label": "C", "text": "Camberwall Market — There's lots to see…" },
  { "label": "D", "text": "Cobbledown Road — A small market…" },
  { "label": "E", "text": "Oldford Lane — Situated in the historic city centre…" },
  { "label": "F", "text": "Purford Market — Close to museums and art galleries…" },
  { "label": "G", "text": "Teddingley Market — Situated under historic city walls…" },
  { "label": "H", "text": "Frome Place — Stalls open during normal daytime shopping hours…" }
],
"questionGroups": [{
  "range": "Questions 6–10",
  "instruction": "For each question, choose the correct answer.",
  "type": "matching-features",
  "features": [
    { "id": "a", "name": "Beckfield Market" },
    { "id": "b", "name": "Rosewell Hill" },
    { "id": "c", "name": "Camberwall Market" },
    { "id": "d", "name": "Cobbledown Road" },
    { "id": "e", "name": "Oldford Lane" },
    { "id": "f", "name": "Purford Market" },
    { "id": "g", "name": "Teddingley Market" },
    { "id": "h", "name": "Frome Place" }
  ],
  "questions": [
    {
      "number": 6,
      "type": "matching-features",
      "prompt": "Jenny wants to buy locally-produced food…",
      "options": [
        { "id": "A", "label": "A" }, { "id": "B", "label": "B" },
        { "id": "C", "label": "C" }, { "id": "D", "label": "D" },
        { "id": "E", "label": "E" }, { "id": "F", "label": "F" },
        { "id": "G", "label": "G" }, { "id": "H", "label": "H" }
      ],
      "answer": "f",
      "explanation": "…"
    }
  ]
}]
```

Quy tắc Part 2:
- `passage[].label` = **một chữ** A–H (không dùng tên market làm label).
- `passage[].text` = `Tên market — mô tả đầy đủ` (dùng em-dash — giữa tên và mô tả).
- `features[]` = **chỉ tên ngắn** (cho pills) — KHÔNG copy full text vào features.
- Phải đủ **8 market** A–H trong passage — không bỏ C/D/E/F.

## Part 4 (Q16–20) — Gapped text + câu A–H ⚠ QUAN TRỌNG

App hiển thị:
- **Cột trái:** bài có gap (16) ........ + block "Sentences" + câu A–H
- **Cột phải:** Gap 16–20 + pills A–H (không lặp danh sách câu)

```json
"passage": [
  { "text": "A new life" },
  { "text": "I used to work as a college lecturer… (16) ........" },
  { "text": "I'd always loved travelling… (17) ........ … (18) ........ …" },
  { "text": "As soon as I got to the island… (19) ........ …" },
  { "text": "In fact I loved it so much… (20) ........ However, …" },
  { "text": "Sentences" },
  { "label": "A", "text": "That's why I knew it was a terrible plan." },
  { "label": "B", "text": "I had trained in icy water in the UK so the crystal clear warm water felt amazing." },
  { "label": "C", "text": "They always ask lots of questions." },
  { "label": "D", "text": "I work far harder than I used to." },
  { "label": "E", "text": "I began joking to friends about sending in an application." },
  { "label": "F", "text": "Afterwards, some people were surprised by my decision but I wasn't too worried." },
  { "label": "G", "text": "I decided I needed a break." },
  { "label": "H", "text": "I needed to explain that first." }
],
"questionGroups": [{
  "type": "matching-features",
  "features": [
    { "id": "a", "name": "That's why I knew it was a terrible plan." },
    { "id": "b", "name": "I had trained in icy water in the UK…" }
  ],
  "questions": [
    {
      "number": 16,
      "type": "matching-features",
      "prompt": "Gap (16)",
      "options": [ /* A–H */ ],
      "answer": "g",
      "explanation": "…"
    }
  ]
}]
```

Quy tắc Part 4:
- `features[].name` = **chỉ nội dung câu** — không prefix "A." hay "A —".
- Đủ **8 câu** A–H trong passage (3 câu thừa không dùng).
- Giữ `(16) ........` … `(20) ........` (8 chấm) trong passage.

## Part 6 (Q27–32) — Open cloze (ONE WORD)

```json
"questionGroups": [{
  "range": "Questions 27–32",
  "instruction": "Write ONE word for each gap.",
  "type": "gap-fill",
  "questions": [
    {
      "number": 27,
      "type": "gap-fill",
      "prompt": "Gap (27)",
      "answer": "every",
      "explanation": "every/each time I've visited."
    }
  ]
}]
```

- Passage giữ `(27) ........` … `(32) ........` (8 chấm).
- `answer`: một từ tiếng Anh, chữ thường.

## Quy tắc bắt buộc

1. Đủ **32 câu** Reading (Q1–32), đánh số liên tục.
2. Chỉ dùng PDF đề + Answer Key — **không bịa**.
3. `passageTitle` mô tả: `"Part 2 — City markets"`, `"Part 4 — A new life"`…
4. Part 2–6: `passage[].text` **bắt buộc** (Highlight).
5. `answer`: matching → `a`–`h`; MC → `a`–`d`; gap-fill → một từ.
6. `explanation`: giải thích ngắn tiếng Việt.
7. Thiếu đáp án → `answer: ""` + `explanation: "⚠ Cần điền từ answer key"`.

## Lỗi thường gặp — tránh

| Lỗi | Cách đúng |
|-----|-----------|
| Part 2: `features[]` copy full text market | features = tên ngắn; full text trong passage label A–H |
| Part 2: thiếu market C–F trong passage | Đủ 8 block label A–H |
| Part 2: `label` = tên market thay vì A–H | `label`: `"A"` … `"H"` |
| Part 4: câu A–H chỉ trong features, không có passage | Thêm passage label A–H sau block "Sentences" |
| Part 4: `features[].name` = `"A. That's why…"` | Chỉ text câu, không prefix chữ cái |
| Part 5: dùng `gap-fill` | Part 5 = `multiple-choice` + 4 options/gap |
| Part 1: gắn đáp án vào passage | Chỉ ảnh; MC ở `questions[]` |
| ZIP có thư mục con | `exam.json` + ảnh cùng cấp |

## Việc làm theo thứ tự

1. Đọc PDF Reading (bỏ Writing) → xác nhận 6 parts · 32 câu.

9. Bảng tóm tắt parts.
