Import từ vựng — Ryan English

Vào đâu?

/app/vocab → chọn bộ (deck) → Nhập từ vựng (icon Upload trên thanh công cụ)

Hoặc: Từ vựng → mở bộ → menu ⋮ → Nhập từ vựng

───

Hỗ trợ gì?

• Từ đơn: abandon, sustainable, hypothesis
• Cụm từ: get off the train, look after, on the other hand
• Cụm động từ: run away, take off, give up

Một thẻ = một mục phrase (dù 1 từ hay nhiều từ).

───

Định dạng file

┌──────────┬──────────┬─────────────────────────────────────────┐
│ Định dạng│ Khuyến nghị │ Ghi chú                                │
├──────────┼──────────┼─────────────────────────────────────────┤
│ .csv     │ Có       │ Nhanh, chỉnh Excel/Google Sheets       │
│ .json    │ Tuỳ chọn │ version 1, có metadata deck (tùy chọn)   │
└──────────┴──────────┴─────────────────────────────────────────┘

Cột CSV bắt buộc:
• phrase — từ hoặc cụm tiếng Anh
• meaning — nghĩa tiếng Việt

Cột tuỳ chọn:
• example — câu ví dụ
• ipaUS — phiên âm
• pos — loại từ (noun, verb, phrasal verb…)

───

Bước import

1. Tạo hoặc chọn bộ đúng chủ đề (IELTS Môi trường, TOEIC Văn phòng…)
2. Bấm Nhập từ vựng
3. Kéo thả file .csv / .json hoặc Chọn file
4. (Tuỳ chọn) Tải file mẫu .csv — có sẵn good, look, run away
5. Xem preview 5 dòng đầu → Import N từ vào deck
6. Học ngay: Lặp lại / Trắc nghiệm / Nghe & Gõ

───

Thêm thủ công

Thêm từ mới → ô Từ / Cụm từ → gõ cả cụm get off the train

───

Xuất ngược

Menu bộ → Xuất CSV hoặc Xuất JSON (backup / chỉnh sửa / chia sẻ)
