## Part 1 (Q1–8) — Multiple-choice cloze

```json
"passage": [
  { "text": "Genealogy is a (0) …..... of history. It concerns family history, (1) ........ than…" }
],
"questionGroups": [{
  "type": "multiple-choice",
  "questions": [
    {
      "number": 1,
      "type": "multiple-choice",
      "prompt": "Gap (1)",
      "options": [
        { "id": "A", "label": "instead" },
        { "id": "B", "label": "rather" },
        { "id": "C", "label": "except" },
        { "id": "D", "label": "sooner" }
      ],
      "answer": "b"
    }
  ]
}]
```

- Giữ `(1) ........` trong passage (8 chấm).
- Example (0) có đáp án trong passage — không tạo câu 0.

## Part 6 (Q37–42) — Gapped text ⚠

Giống PET B1 Part 4:

```json
"passage": [
  { "text": "…(37) ........…" },
  { "text": "Sentences" },
  { "label": "A", "text": "Through endless tries at the usual exercises…" },
  { "label": "G", "text": "It takes at least a decade…" }
],
"questionGroups": [{
  "type": "matching-features",
  "features": [
    { "id": "a", "name": "Through endless tries…" }
  ],
  "questions": [
    {
      "number": 37,
      "type": "matching-features",
      "prompt": "Gap (37)",
      "options": [ /* A–G */ ],
      "answer": "d"
    }
  ]
}]
```

- `features[].name` = chỉ nội dung câu (không prefix "A.").
- Đủ 7 câu A–G; 6 gap 37–42; 1 câu thừa.
