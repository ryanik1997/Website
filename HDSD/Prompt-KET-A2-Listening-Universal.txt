## Part 2 (Q6–10) — Gap-fill form ⚠ KHÔNG double form

**CẤM** nhét cả form (có `(6)` `(7)` …) vào `instruction` **và** lặp lại label trong `questions[].prompt` → app hiện **double**.

### Đúng

```json
{
  "partNumber": 2,
  "rangeLabel": "Questions 6–10",
  "instruction": "You will hear a phone message about going to play a hockey match.\n\nName of other team: Tigers",
  "passageTitle": "Hockey match",
  "audioFile": "listening.mp3",
  "questions": [
    {
      "number": 6,
      "type": "gap-fill",
      "prompt": "Day:",
      "answer": "Saturday",
      "wordLimit": 1,
      "explanation": "…"
    },
    {
      "number": 7,
      "type": "gap-fill",
      "prompt": "Name of village:",
      "answer": "Newton",
      "wordLimit": 1
    },
    {
      "number": 8,
      "type": "gap-fill",
      "prompt": "Time guests arrive:",
      "answer": "8/eight",
      "acceptableAnswers": ["8", "eight"],
      "wordLimit": 1,
      "explanation": "Answer key: 8 / eight — app chấp nhận cả hai."
    }
  ]
}
```

| Trường | Nội dung |
|--------|----------|
| `instruction` | Câu "You will hear…" + **chỉ** dòng tĩnh đã điền sẵn (vd. `Name of other team: Tigers`) |
| `passageTitle` | Tiêu đề form (vd. `Hockey match`) — **không** lặp trong instruction |
| `questions[].prompt` | **Chỉ label** gap: `"Day:"`, `"Travel to village by:"` — **không** ghi `(6)` hay cả form |

### Sai (double UI)

```json
"instruction": "You will hear…\n\nHockey match\nDay: (6)\nName of village: (7)\n…"
"prompt": "Day:"
```

→ App vừa vẽ form trong instruction vừa vẽ input `Day:` → **double**.

- ONE word / number / date mỗi gap
- App import vẫn **tự cắt** form `(N)` khỏi instruction nếu DeepSeek nhét nhầm; prompt đúng vẫn tốt hơn

## ⚠ App 2026-07 — Gap-fill đáp án kép + WebP

Chi tiết đầy đủ: **HDSD/Prompt-Listening-Cambridge.txt** (§ Gap-fill · § Ảnh WebP).

### Gap-fill — app chấm cả biến thể

| Viết trong JSON | User gõ → đúng |
|-----------------|----------------|
| `"answer": "8/eight"` | `8` hoặc `eight` |
| `"answer": "8", "acceptableAnswers": ["eight"]` | idem |
| `"answer": "could/may"` | `could` hoặc `may` |
| `"answer": "8 OR eight"` | idem |

```json
{
  "number": 6,
  "type": "gap-fill",
  "prompt": "Time guests arrive:",
  "answer": "8/eight",
  "acceptableAnswers": ["8", "eight"],
  "wordLimit": 1,
  "explanation": "Theo Answer key: 8 / eight."
}
```

- Bỏ ngoặc key: `"(August)"` → `"August"`; số `(19)` → `"19"` hoặc `"19/nineteen"`
- **Không** `"answer": "B (best)"` — chỉ letter
- Review hiện **「8 hoặc eight」**

### Ảnh — WebP

- File: `q1.webp`, `q2.png`, `part7-page.webp` … OK
- JSON có thể ghi `"imageFile": "q1.jpg"` dù file là **q1.webp** (app map basename)
- Part 1 composite: `q1.jpg|webp` … (KET 5 ảnh / PET 7 ảnh)

### Audio import

- `listening.mp3` trong ZIP gắn local blob — **không** bị đè audio catalog Test 1 khi title là Test 2/3/4
- Title bắt buộc: `… Book {B} — Test {T}`

## Việc làm

1. Book + Test từ thư mục/PDF
2. Map 5 parts · 25 câu
3. Crop q1.jpg|webp … q5.jpg|webp (Part 1)
4. Answer key → `answer` (`8/eight` nếu kép) + `acceptableAnswers` (tuỳ) + `explanation`
5. Audioscript (PDF text hoặc `audioscript.txt`) → `ttsText` / để app auto gắn
6. JSON.parse PASS → ZIP (exam + mp3 + ảnh + key/script)
7. Bảng tóm tắt Part | Câu | Type | Media | Đáp án
