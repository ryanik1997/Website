## ⚠ SCHEMA BẮT BUỘC — KHÔNG ĐỔI TÊN FIELD

App CHỈ đọc đúng schema sau. KHÔNG dùng: part, title, questions, id, question, options.label+text.

Mỗi part PHẢI có:
- partNumber (KHÔNG phải "part")
- rangeLabel (vd "Questions 1–6")
- passageTitle
- passage[] (mảng object, KHÔNG phải string)
- questionGroups[] (MẢNG — KHÔNG đặt questions ở ngoài)

Mỗi câu trong questionGroups[].questions PHẢI có:
- number (KHÔNG phải "id")
- type
- prompt (KHÔNG phải "question")
- options: [{ "id": "A", "label": "..." }] (KHÔNG dùng label+text)
- answer, explanation

Schema exam.json:
{
  "version": 1,
  "title": "KET A2 Reading — Test 3",
  "durationMinutes": 60,
  "bandHint": "A2 Key Reading",
  "examTrack": "cambridge",
  "cambridgeLevel": "a2",
  "parts": [ /* 5 parts, 30 câu */ ]
}

## 5 parts (30 câu)

| Part | Câu | Type | Passage |
|------|-----|------|---------|
| 1 | 1–6 | multiple-choice | 6 ảnh part1-q1…q6 |
| 2 | 7–13 | multiple-choice | text + label tên người |
| 3 | 14–18 | multiple-choice | passageTitle + passageSubtitle + thân bài phân đoạn |
| 4 | 19–24 | multiple-choice | gapped text (19) ........ — 3 từ A/B/C, KHÔNG gap-fill |
| 5 | 25–30 | gap-fill | email + Example: 0 are — ONE WORD |

## Part 1 — Signs
passage = 6 block imageFile (part1-q1.jpg … part1-q6.jpg)
questions 1–6: type multiple-choice, options A/B/C, answer a/b/c

## Part 2 — Matching people
passage: { "label": "Tên", "text": "..." } — 1 block / người
questions 7–13: MC, options = tên người A/B/C — KHÔNG dùng matching-features

## Part 3 (Q14–18) — Reading comprehension

PDF có khung đen chứa: tiêu đề bold → subtitle italic → các đoạn thân bài.

| PDF (trong khung) | JSON |
|-------------------|------|
| Tiêu đề bold | passageTitle |
| Dòng italic phụ | passageSubtitle |
| Mỗi đoạn thân bài | passage[] — 1 block / đoạn |

SAI:
- passageTitle = "Part 3" hoặc "PART 3 — QUESTIONS 14–18"
- Gộp title + subtitle + body vào 1 text với \n\n

ĐÚNG:
{
  "partNumber": 3,
  "rangeLabel": "Questions 14–18",
  "passageTitle": "[tiêu đề bold trong khung PDF]",
  "passageSubtitle": "[dòng italic dưới tiêu đề, nếu có]",
  "passage": [
    { "text": "[đoạn 1 — copy nguyên văn]" },
    { "text": "[đoạn 2]" },
    ...
  ],
  "questionGroups": [{
    "range": "Questions 14–18",
    "instruction": "For each question, choose the correct answer.",
    "type": "multiple-choice",
    "questions": [ /* 5 câu 14–18, MC A/B/C */ ]
  }]
}

## Part 4 — Multiple-choice cloze ⚠ QUAN TRỌNG
questionGroups.type = "multiple-choice" (KHÔNG gap-fill)
Mỗi câu 19–24: 3 options A/B/C (copy 3 từ từ PDF)
passage giữ marker (19) ........
passageTitle = tiêu đề bài trong PDF (vd "The Nobel Prize")

## Part 5 — Email open cloze
questionGroups.type = "gap-fill"
passage: 1 block / email (From/To), giữ Example (0) are và (25)…(30) ........
questions 25–30: gap-fill ONE WORD
