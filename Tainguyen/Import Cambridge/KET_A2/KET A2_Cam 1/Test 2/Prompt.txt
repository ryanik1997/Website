## ⚠ SCHEMA BẮT BUỘC — KHÔNG ĐỔI TÊN FIELD

App CHỈ đọc đúng schema sau. KHÔNG dùng: part, title, questions, id, question, options.label+text.

Mỗi part PHẢI có:
- partNumber (KHÔNG phải "part")
- rangeLabel (vd "Questions 1–6")
- passageTitle
- passage[] (mảng object, KHÔNG phải string)
- questionGroups[] (MẢNG — KHÔNG đặt questions ở ngoài)

Mỗi câu trong questionGroups[].questions PHẢI có:
- number (KHÔNG phải "id")
- type
- prompt (KHÔNG phải "question")
- options: [{ "id": "A", "label": "..." }] (KHÔNG dùng label+text)
- answer, explanation

Ví dụ Part 1 đúng:
{
  "partNumber": 1,
  "rangeLabel": "Questions 1–6",
  "passageTitle": "Part 1 — Signs and messages",
  "passage": [{ "imageFile": "part1-q1.jpg" }, ...],
  "questionGroups": [{
    "range": "Questions 1–6",
    "instruction": "For each question, choose the correct answer.",
    "type": "multiple-choice",
    "questions": [
      {
        "number": 1,
        "type": "multiple-choice",
        "prompt": "Contact Louise if you want to",
        "options": [
          { "id": "A", "label": "improve the way the station looks." },
          { "id": "B", "label": "learn how to make a nice garden." },
          { "id": "C", "label": "travel to see some beautiful gardens." }
        ],
        "answer": "a",
        "explanation": "..."
      }
    ]
  }]
}

Schema exam.json gốc:

}

## Part 3 (Q14–18) — Reading comprehension ⚠ BẮT BUỘC GIỐNG Part3.jpg

Tham chiếu layout: Part3.jpg (khung đen trong PDF).

### Cách PDF hiển thị (app tái tạo như sau)
| Vị trí PDF | Field JSON | Hiển thị app |
|------------|------------|--------------|
| PART 3 / QUESTIONS 14–18 | rangeLabel | "Questions 14–18" (cột trái) |
| For each question, choose… | questionGroups[].instruction | cột phải (câu hỏi) |
| **My life as a tour guide** (bold, trong khung) | passageTitle | tiêu đề bold cột trái |
| *Kerri Benson talks…* (italic, trong khung) | passageSubtitle | dòng italic dưới tiêu đề |
| 5 đoạn thân bài (trong khung) | passage[] — 5 block text | 5 đoạn văn riêng |

### SAI — KHÔNG làm
- passageTitle = "Part 3 — Reading comprehension" hoặc "PART 3 — QUESTIONS 14–18"
- Gộp title + subtitle + thân bài vào 1 block text với \n\n
- Đặt title/subtitle vào passage[] thay vì passageTitle / passageSubtitle

### Quy tắc Part 3 (mọi đề KET)

2. passageSubtitle = dòng phụ italic ngay dưới tiêu đề (nếu PDF có)


- Mỗi đoạn PDF = 1 block — KHÔNG gộp \n\n trong 1 block
- questionGroups.type = "gap-fill"; questions 25–30 ONE WORD

### 2 email (Test 1) — answer key có 2 phần
