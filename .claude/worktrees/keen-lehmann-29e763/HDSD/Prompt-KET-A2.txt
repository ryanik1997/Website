## Part 3 (Q14–18) — Reading comprehension ⚠ layout khung đen PDF

PDF Part 3 có **khung đen** chứa: tiêu đề bold → subtitle italic → 5 đoạn thân bài.
App hiển thị giống khi tách đúng field:

| PDF (trong khung) | JSON |
|-------------------|------|
| Tiêu đề bold (vd "My life as a tour guide") | `passageTitle` |
| Dòng italic phụ (vd "Kerri Benson talks…") | `passageSubtitle` |
| Mỗi đoạn thân bài | `passage[]` — **1 block / 1 đoạn** |

**SAI:**
- `passageTitle` = "Part 3 — Reading comprehension" hoặc "PART 3 — QUESTIONS 14–18"
- Gộp title + subtitle + body vào 1 `text` với `\n\n`
- Đặt title/subtitle vào `passage[]`

**ĐÚNG:**
```json
"partNumber": 3,
"rangeLabel": "Questions 14–18",
"passageTitle": "My life as a tour guide",
"passageSubtitle": "Kerri Benson talks about the job she loves.",
"passage": [
  { "text": "Đoạn 1 — copy nguyên văn…" },
  { "text": "Đoạn 2 — copy nguyên văn…" },
  { "text": "Đoạn 3…" },
  { "text": "Đoạn 4…" },
  { "text": "Đoạn 5…" }
],
"questionGroups": [{
  "range": "Questions 14–18",
  "instruction": "For each question, choose the correct answer.",
  "type": "multiple-choice",
  "questions": [ /* 5 câu 14–18, MC A/B/C */ ]
}]
```

- 5 câu MC, mỗi câu 3 options A/B/C (copy nguyên văn từ PDF).

## Part 4 (Q19–24) — Multiple-choice cloze ⚠ QUAN TRỌNG

Đề gốc KET Part 4: mỗi gap chọn **1 trong 3 từ** (A/B/C).
App hiển thị gap ở cột trái; học sinh chọn từ ở cột phải.

**SAI** (app báo lỗi / không làm được):
```json
{ "number": 19, "type": "gap-fill", "prompt": "Gap (19)", "answer": "short" }
```

**ĐÚNG**:
```json
"passage": [
  { "text": "The coldest place to live" },
  { "text": "…During the winter, the days are (19) ........, and temperatures…" }
],
"questionGroups": [{
  "range": "Questions 19–24",
  "instruction": "For each question, choose the correct answer.",
  "type": "multiple-choice",
  "questions": [
    {
      "number": 19,
      "type": "multiple-choice",
      "prompt": "Gap (19)",
      "options": [
        { "id": "A", "label": "early" },
        { "id": "B", "label": "short" },
        { "id": "C", "label": "quick" }
      ],
      "answer": "b",
      "explanation": "Mùa đông ngày ngắn (short)."
    }
  ]
}]
```

- `questionGroups[].type` = `"multiple-choice"` (không phải gap-fill).
- Mỗi câu 19–24 có đủ 3 `options` — copy 3 từ từ PDF.
- Passage giữ marker `(19) ........` (8 chấm) — app tự chuẩn hóa hiển thị.

## Part 5 (Q25–30) — Email open cloze

Layout PDF: From/To (header) → các đoạn thân email → "Write soon!"

**1 email** (vd Erin → Lara):
```json
"passageTitle": "From Erin to Lara",
"passage": [
  { "label": "From", "text": "Erin" },
  { "label": "To", "text": "Lara" },
  { "text": "How (0) are you? … (25) ........ …" },
  { "text": "It's taken me… (28) ........ … (30) ........ …" },
  { "text": "Write soon!" }
]
```

**2 email** (vd Test 1 Sylvia ↔ Felix) — answer key có 2 phần, gaps 25–27 + 28–30:
```json
"passage": [
  { "text": "From: Sylvia\nTo: Felix\n\n…(25)…(26)…(27)…" },
  { "text": "From: Felix\nTo: Sylvia\n\n…(28)…(29)…(30)…" }
]
```

**Đáp án kép** (answer key ghi 2 từ chấp nhận): `"answer": "a/one"` hoặc `"would/will"` — app chấp nhận cả hai khi chấm.

- **Phân đoạn:** mỗi đoạn thân email = 1 block — KHÔNG gộp `\n\n` (app không tách đoạn trong 1 block).
- Giữ Example `(0) are` — **không** tạo câu hỏi số 0.
- Gaps: `(25) ........` … `(30) ........` (8 chấm).
- `answer`: một từ (hoặc `từ1/từ2` nếu key có 2 đáp án), chữ thường.

## Quy tắc bắt buộc

1. Đủ **30 câu** Reading (Q1–30), đánh số liên tục — không nhảy số.
2. Chỉ dùng nội dung PDF đề + Answer Key — **không bịa**.
3. `passageTitle` mô tả nội dung: `"Part 1 — Signs and messages"`, `"Part 4 — The coldest place to live"` — không chỉ `"PART 4 — QUESTIONS …"`.
4. Part 2–5: `passage[].text` **bắt buộc** (Highlight khi làm bài).
5. **Phân đoạn theo PDF:** PDF có bao nhiêu đoạn → `passage[]` có bấy nhiêu block. **KHÔNG** gộp nhiều đoạn vào 1 `text` với `\n\n`. Part 3: title/subtitle → `passageTitle` + `passageSubtitle`; thân bài mỗi đoạn = 1 block.
6. `answer`: MC → `a`/`b`/`c`; gap-fill → một từ tiếng Anh thường.
7. `explanation`: giải thích ngắn tiếng Việt (tuỳ chọn nhưng nên có).
8. Mỗi câu có `type` khớp `questionGroups[].type`.
9. Thiếu đáp án → `answer: ""` + `explanation: "⚠ Cần điền từ answer key"`.

## Lỗi thường gặp — tránh

| Lỗi | Cách đúng |
|-----|-----------|
| Part 4 dùng `gap-fill` | Part 4 = `multiple-choice` + 3 options/câu |
| Part 2 dùng `matching-features` | Part 2 = MC, options là tên người |
| Part 1 gắn đáp án vào passage | Chỉ ảnh; MC ở `questions[]` |
| Passage Part 2+ để trống | Copy text từ PDF |
| Gộp nhiều đoạn PDF vào 1 block `\n\n` | Mỗi đoạn PDF = 1 block `passage[]` |
| ZIP có thư mục con | `exam.json` + ảnh cùng cấp |
| Nhầm Writing Part 6–7 vào Reading | Chỉ Parts 1–5 Reading |
