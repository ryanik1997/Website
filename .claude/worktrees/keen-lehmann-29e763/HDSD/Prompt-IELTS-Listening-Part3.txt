## Schema Part 3

```json
{
  "partNumber": 3,
  "rangeLabel": "Questions 21–30",
  "instruction": "…",
  "passageTitle": "…",
  "audioFile": "listening.mp3",
  "notePassageLayout": "form",
  "notePassageSections": [ /* c1, c2 — nhiều khối note */ ],
  "noteTables": [ /* c2, c7 — bảng */ ],
  "questions": [ /* 10 câu, thứ tự 21→30 */ ]
}
```

## Thứ tự `questions[]` (app tách segment)

```
gap-fill → multiple-choice → matching → choose-two → flowchart
```

**Ví dụ c1:** MC 21–22 → gap 23–25 → gap 26–30  
**Ví dụ c5:** Choose TWO 21–22 → Choose TWO 23–24 → matching 25–30  
**Ví dụ c6:** MC 21–25 → flowchart matching 26–30

## Section header (câu ĐẦU mỗi nhóm)

```json
"sectionRange": "Questions 21 – 22",
"sectionInstruction": "Choose the correct letter A, B or C.",
"sectionTitle": "THE LATER LIFE OF THOR HEYERDAHL"
```

## notePassageSections[] (c1, c2 — nhiều khối note tách)

```json
"notePassageSections": [
  {
    "gapNumbers": [23, 24, 25],
    "instruction": "Complete the sentences below. Write ONE WORD ONLY for each answer.",
    "blocks": [
      { "type": "static", "text": "Kira says that lecturers are easier to" },
      { "type": "gap", "number": 23 },
      { "type": "static", "text": "than those in her home country." }
    ]
  },
  {
    "gapNumbers": [26, 27, 28, 29, 30],
    "instruction": "Complete the notes below. Write NO MORE THAN THREE WORDS…",
    "blocks": [ /* … */ ]
  }
]
```

- Mỗi section có `gapNumbers` khớp câu gap trong `questions`
- Không trộn gap của 2 khối vào 1 `notePassage` dài (dùng sections)

## Choose TWO — "Which TWO …?" (c3, c5)

**Tham chiếu:** `Giaodien/two-choice.jpg` · mẫu `p3-c3-template.json`

Trên Answer Key (Cam20 Test 1 P3) — **3 nhóm** Choose TWO:
| Câu | Answer Key | exam.json (cả 2 câu nhóm) |
|-----|------------|---------------------------|
| 21–22 | 21 C/E, 22 C/E | `"answer": "C/E"` |
| 23–24 | 23 A/C, 24 A/C | `"answer": "A/C"` |
| 25–26 | 25 A/B, 26 A/B | `"answer": "A/B"` |
| 27–30 | 27 A, 28 B… | MC — không Choose TWO |

**UI app:** giống Part 2 — 5 options A–E + 2 ô, mỗi ô pill A B C D E.
Mỗi ô đúng nếu chọn một trong hai chữ cái đúng. Thứ tự giữa 2 ô không bắt buộc.

**KHÔNG** nhầm với letter matching (c4, c5, c7) — Choose TWO: prompt "Which TWO…",
đáp án slash `"C/E"`; letter matching: prompt item ngắn, đáp án 1 chữ `"G"`.

```json
{
  "number": 21,
  "type": "matching",
  "prompt": "Which TWO things do the students both believe are responsible for the increase in loneliness?",
  "options": [
    { "id": "A", "label": "…" },
    { "id": "B", "label": "…" },
    { "id": "C", "label": "…" },
    { "id": "D", "label": "…" },
    { "id": "E", "label": "…" }
  ],
  "answer": "C/E",
  "explanation": "…",
  "sectionRange": "Questions 21 – 22",
  "sectionInstruction": "Choose TWO letters, A–E."
},
{
  "number": 22,
  "type": "matching",
  "prompt": "Which TWO things do the students both believe are responsible for the increase in loneliness?",
  "options": [ /* GIỐNG HỆT câu 21 */ ],
  "answer": "C/E",
  "explanation": "Cặp đáp án 21–22."
}
```

Quy tắc:
- **2 câu** `matching` **liên tiếp**, **cùng prompt**, **cùng options**
- `answer` **giống nhau** trên cả 2 câu — đọc Answer Key (21 C/E → `"C/E"`)
- c5 có **2 nhóm** Choose TWO (21–22, 23–24) trước letter matching 25–30
- `options`: nhãn đầy đủ — KHÔNG `"label": "A"`

## Letter matching (c4, c5, c7) — KHÁC Choose TWO

Giống Part 2 — options A–G giống nhau, prompt = item ngắn.

## Flow-chart (c6)

```json
{
  "number": 26,
  "type": "matching",
  "prompt": "",
  "options": [
    { "id": "A", "label": "bullet points" },
    { "id": "B", "label": "film" }
  ],
  "answer": "G",
  "flowChart": true,
  "flowChartEnd": "He'll write the complete paper.",
  "gapLead": "He'll read",
  "gapTrail": "and choose his topic.",
  "sectionRange": "Questions 26 – 30",
  "sectionInstruction": "Complete the flow-chart below…",
  "sectionTitle": "How James will write his paper on the Vikings"
}
```

- `flowChart: true` trên câu đầu nhóm (26)
- `flowChartEnd` chỉ trên câu 26
- Mỗi bước: `gapLead` + select + `gapTrail`
- Câu 27–30: `flowChart: true`, cùng options A–G
