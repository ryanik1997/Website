## Schema Part 1

```json
{
  "partNumber": 1,
  "rangeLabel": "Questions 1–10",
  "instruction": "Complete the form below. Write NO MORE THAN THREE WORDS AND/OR A NUMBER for each answer.",
  "passageTitle": "JOB INQUIRY",
  "audioFile": "listening.mp3",
  "notePassageLayout": "form",
  "notePassage": [ /* hoặc noteTable / noteTables */ ],
  "questions": [ /* 10 câu, đúng thứ tự 1→10 */ ]
}
```

## Thứ tự `questions[]` (app tách segment tự động)

```
gap-fill (1,2,3…) → multiple-choice → matching (Choose TWO) → gap-fill tiếp
```

Ví dụ a4: câu 1–4 gap → 5–6 Choose TWO → 7–10 gap.

## notePassage blocks

| type      | Ý nghĩa                          |
|-----------|----------------------------------|
| static    | Dòng chữ không có ô trống        |
| section   | Tiêu đề mục (Extra benefits…)    |
| example   | Dòng Example (dùng với layout form) |
| gap       | Ô trống — number = questions[].number |

Mỗi câu `gap-fill` PHẢI có đúng 1 `gap` trong notePassage hoặc noteTable.

## noteTable / noteTables[]

```json
"notePassageLayout": "table",
"noteTable": {
  "headers": ["Cột 1", "Cột 2", …],
  "rows": [{ "cells": [[{ "type": "static", "text": "…" }, { "type": "gap", "number": 1 }]] }]
}
```

Cell: `static` | `gap` | `break` (xuống dòng trong cùng ô).

Nhiều bảng → `noteTables[]`, mỗi bảng có `gapNumbers: [1,2,3,4]` bắt buộc.

## gap-fill question

```json
{
  "number": 1,
  "type": "gap-fill",
  "prompt": "Type of work:",
  "options": [],
  "answer": "answering phone",
  "wordLimit": 3,
  "gapLead": "Type of work:",
  "gapTrail": "of an international company",
  "explanation": "Trả lời điện thoại."
}
```

`gapLead` / `gapTrail` = chữ trước/sau ô (inline). Không dùng noteBefore thay notePassage.

## multiple-choice

```json
{
  "number": 1,
  "type": "multiple-choice",
  "prompt": "…",
  "options": [
    { "id": "A", "label": "…" },
    { "id": "B", "label": "…" },
    { "id": "C", "label": "…" }
  ],
  "answer": "B",
  "explanation": "…"
}
```

## Choose TWO — "Which TWO …?" (a4 câu 5–6)

**Tham chiếu:** `Giaodien/two-choice.jpg` (UI + Answer Key A/E) · mẫu `p1-mixed-a4-template.json`

Trên đề PDF: một câu hỏi "Which TWO …?", 5 lựa chọn A–E, **2 ô trả lời** (câu 5 và 6).
Trên Answer Key: `5 A/E`, `6 A/E` → mỗi ô đúng nếu chọn **A hoặc E** (thứ tự không bắt buộc).

**UI app:** danh sách A–E (checkbox trang trí) + 2 hàng số 5 và 6, mỗi hàng pill A B C D E.

**KHÔNG** dùng `type: "multiple-choice"` cho Choose TWO — phải `type: "matching"`.

```json
{
  "number": 5,
  "type": "matching",
  "prompt": "Which TWO of the following are offered free of charge at Shore Lane Health Centre?",
  "options": [
    { "id": "A", "label": "acupuncture" },
    { "id": "B", "label": "employment medicals" },
    { "id": "C", "label": "sports injury therapy" },
    { "id": "D", "label": "travel advice" },
    { "id": "E", "label": "vaccinations" }
  ],
  "answer": "A/E",
  "explanation": "Hai dịch vụ miễn phí — A và E.",
  "sectionRange": "Questions 5 – 6",
  "sectionInstruction": "Choose TWO letters, A–E."
},
{
  "number": 6,
  "type": "matching",
  "prompt": "Which TWO of the following are offered free of charge at Shore Lane Health Centre?",
  "options": [ /* GIỐNG HỆT câu 5 */ ],
  "answer": "A/E",
  "explanation": "Cặp đáp án 5–6: A và E."
}
```

Quy tắc:
- **2 câu** `matching` **liên tiếp**, **cùng prompt**, **cùng options**
- `answer` **giống nhau** trên cả 2 câu (vd. `"A/E"`, `"B/D"`)
- `options`: nhãn đầy đủ — KHÔNG `"label": "A"` (UI hiện "A A")
- Choose TWO **không** cần `notePassage` / `noteTable` (khác gap-fill)
- `sectionRange` + `sectionInstruction` trên **câu đầu** nhóm (câu 5)

**Đọc Answer Key → ghi JSON:**
| Answer Key | exam.json (cả 2 câu) |
|------------|----------------------|
| 5 A/E, 6 A/E | `"answer": "A/E"` |
| 5 B/D, 6 B/D | `"answer": "B/D"` |
